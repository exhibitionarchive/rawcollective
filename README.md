# 原质社 RAW COLLECTIVE Website

This is a local static website for 原质社 RAW COLLECTIVE.

Open `index.html` in a browser to preview it. Edit `members.js` to add real member profiles.

## Content Shape

Each member includes:

- English and Traditional Chinese name
- English and Traditional Chinese role
- One-paragraph bio in both languages
- CV sections for projects, exhibitions, prizes, or other categories
- Links

Current member content is drafted from the Desktop folder `原质社 RAW COLLECTIVE`.
